⚡ SignalHire Extractor Guide
How do I load PDFs? +
Click the drop zone or drag & drop your SignalHire PDF files directly. The tool automatically detects the contact person name from the filename (e.g. "John Smith.pdf" → "John Smith").
What is the Default Folder feature? +
You can set a default folder path (stored in localStorage) so the app remembers your SignalHire PDF folder between sessions.
What does Confidence % mean? +
Confidence is scored from 0–100% based on how many fields were successfully extracted: name (20%), company (20%), position (15%), phone (15%), email (15%), LinkedIn (15%).
Why are some phone fields empty? +
Phone validation requires at least 7 digits and rejects strings with long alphabetic runs. Only genuine phone numbers pass validation.
🔍 Data Comparator Guide
How does the matching algorithm work? +
The comparator uses fuzzy string matching (Levenshtein distance, partial ratio, token sort, token set) combined with the Hungarian assignment algorithm for optimal 1-to-1 matching.
What is the Match Threshold? +
The threshold (50–99) is the minimum similarity score for two company names to be considered a match. 75 is a good default.
What does the tool strip from company names? +
Words like PRIVATE, LIMITED, LTD, PVT, LLP, INC, CORPORATION, CORP, INDIA, INDIAN, THE, MR, MS etc. are removed before comparison for cleaner matching.
Only today's files are showing — why? +
The file scanner filters to only show .xlsx files modified today. Use "Manual .xlsx browse" to open any file regardless of modification date.
❓ Frequently Asked Questions
Does this tool save my data anywhere? +
No. All processing happens locally in your browser. No data is sent to any server. localStorage is only used for settings and history metadata.
Can I use this offline? +
Yes, once the page is loaded with its CDN dependencies. The core functionality runs entirely client-side.
What browsers are supported? +
Chrome 86+ and Edge 86+ for full folder access features. Firefox and Safari will fall back to manual file selection.
🔧 Troubleshooting
PDFs are loading but fields are empty +
Ensure your PDFs are genuine SignalHire exports. The parser uses specific column positions. Scanned/image PDFs without text layers won't work.
Folder access was blocked +
Browser security may block automatic folder access. Click "Open PyCharm Folder" and explicitly grant permission. Permissions are stored via IndexedDB.
Large files cause the browser to hang +
Processing uses async/await with batched operations. For very large datasets (1000+ rows), the Hungarian algorithm may take 10–30 seconds. The progress bar will indicate status.
